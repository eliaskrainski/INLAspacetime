# The 2nd order temporal matrices with boundary correction

The 2nd order temporal matrices with boundary correction

## Usage

``` r
Jmatrices(tmesh)
```

## Arguments

- tmesh:

  Temporal mesh

## Value

return a list of temporal finite element method matrices for the
supplied mesh.

## Details

Temporal GMRF representation with stationary boundary conditions as in
Appendix E in Lindgren et. al. (2024).
